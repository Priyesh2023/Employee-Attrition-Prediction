# Employee Attrition Prediction using Machine Learning

## Project Overview
This project predicts employee attrition using the IBM HR Analytics dataset.

## Objectives
- Identify employees at risk of leaving
- Analyze key attrition factors
- Compare multiple machine learning models
- Generate HR-focused business recommendations

## Dataset
IBM HR Analytics Employee Attrition Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Models Used
1. Logistic Regression
2. Random Forest Classifier
3. Gradient Boosting Classifier

## Results
| Model | Precision | Recall | F1 Score | ROC-AUC |
|---------|---------|---------|---------|---------|
| Logistic Regression | 0.341 | 0.617 | 0.439 | 0.695 |
| Random Forest | 0.571 | 0.085 | 0.148 | 0.536 |
| Gradient Boosting | 0.588 | 0.213 | 0.313 | 0.592 |

## Key Insights
- Sales department had the highest attrition rate.
- Sales Representatives were the most vulnerable employee group.
- Lower income employees showed higher attrition.
- Poor work-life balance increased attrition risk.
- Employees with shorter tenure were more likely to leave.
